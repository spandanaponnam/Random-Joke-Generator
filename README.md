# Random Joke Generator

A simple Python script that prints a random joke using the [PyJokes](https://pypi.org/project/pyjokes/) library. Perfect for a quick laugh and to practice Python basics.

## Features
- Fetches jokes in English.
- Randomly selects a joke from the available set.
- Easy to run and customize.

## Installation

1. Make sure you have Python installed (version 3.x).  
2. Install the required library:
```bash
pip install pyjokes
